## Краткое описание
Во время движения этот игрок может использовать Pogo, чтобы перепрыгнуть через одну соседнюю клетку, как Jump, игнорируя обычные отрицательные модификаторы прыжка. Эта черта не может сочетаться с Leap.

Полная формулировка: https://bloodbowlbase.ru/bb2025/core_rules/skills_and_traits/

## Изменения Gata League
- Во время движения этого игрока он может использовать Pogo, чтобы перепрыгнуть через одну соседнюю клетку. Pogo можно использовать только один раз за активацию.
