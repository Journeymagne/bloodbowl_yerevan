## Краткое описание
Этот игрок может выполнить Block по двум разным соседним соперникам одновременно, уменьшив свой ST на 2 для этих блоков. Он не может выполнить Follow-up, и этот навык нельзя сочетать с Frenzy.

Полная формулировка: https://bloodbowlbase.ru/bb2025/core_rules/skills_and_traits/

## Изменения Gata League
- Этот игрок может использовать Multiple Block, чтобы затронуть двух соседних соперников как обычно. Multiple Block также работает со специальными действиями, заменяющими Block.
