
|  *2d6*   | *Событие*                                                                                                                                                                                                                |
| :------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ***2***  | **Взятки:** Каждая команда получает по одной Взятке. Эта взятка сгорает в конце игры                                                                                                                                     |
| ***3***  | **Таймаут:** Если номер хода 4+, маркер хода двигается на одно деление назад. В ином случае, передвиньте маркер хода на одно деление вперед.                                                                             |
| ***4***  | **Надёжная защита:** d3+1 Открытых игроков выбивающей команды могут изменить свою расстановку, следуя стандартным правилам.                                                                                              |
| ***5***  | **Хай-кик:** Выберите одного Открытого игрока принимающей команды и передвиньте в точку приземления мяча.                                                                                                                |
| ***6***  | **Ликующие фанаты:** Оба тренера кидают d6 и добавляют к значению броска количество своих чирлидеров. Победитель получает +1 к Силе игрока во время своего следующего Блока. В случае ничьей бонус получают оба тренера. |
| ***7***  | **Идеальная подготовка:** Оба тренера кидают d6 и добавляют к значению броска количество своих помощников . Победитель получает Командный переброс до конца драйва. В случае ничьей его получают оба тренера.            |
| ***8***  | **Смена погоды:** Киньте d6 и сопоставьте его с таблицей погоды. Если погода нормальная, мяч смещается 3 раза в воздухе.                                                                                                 |
| ***9***  | **Снэп:** d3+1 Открытых игроков принимающей команды могут передвинуться на одну клетку в любом направлении.                                                                                                              |
| ***10*** | **Блиц:** 1 Открытый игрок выбивающей команды может немедленно совершить Движение, Блиц или Бросок товарища.                                                                                                             |
| ***11*** | **Драка с судьёй:** Оба тренера выбирают 3-х игроков, после чего случайно выбранный игрок из этого числа в каждой команде удаляется до конца тайма.                                                                      |
| ***12*** | **Вторжение на поле:** Оба игрока выбирают случайного игрока в своей команде, и он получает состояние Оглушен.                                                                                                           |
