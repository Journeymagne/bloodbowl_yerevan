Когда этот игрок пытается Уклониться (Dodge), он не получает отрицательных модификаторов к своей проверке Ловкости за то, что Отмечен (Marked) игроками противника.

Кроме того, этот игрок применяет модификатор -1 к проверке Ловкости при попытке Перехватить мяч.

Игрок с этим свойством более склонен к травмам, поэтому если для него делается Бросок на Травму, вместо этого бросайте по Таблице Травм Коротышек:

| 2D6   | РЕЗУЛЬТАТ                                                                                                                                               |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **2‑6**   | **ОГЛУШЕН:** Игрок оглушен                                                                                                                              |
| **7‑8**   | **НОКАУТ:** Игрок нокаутирован, переместите его в соответствующую ячейку раздевалки.                                                                    |
| **9**     | **РАНЕН:** Игрок травмирован, переместите его в соответствующую ячейку раздевалки, без броска на Серьезную Травму.                                      |
| **10‑12** | **ТРАВМА:** Игрок травмирован, переместите его в соответствующую ячейку раздевалки. Ваш оппонент делает бросок на Серьезную Травму против этого игрока. |
