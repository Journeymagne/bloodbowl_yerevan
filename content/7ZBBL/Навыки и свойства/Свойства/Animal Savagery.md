Каждый раз, когда этот игрок активируется, после объявления своего действия он должен бросить D6. Он может применить модификатор +2 к броску, если объявил действия Блок или Блиц. При результате 4+ игрок может выполнить заявленное действие как обычно.

При результате 1–3 этот игрок набрасывается на одного из своих товарищей по команде. Выберите одного Стоящего на ногах товарища по команде, соседнего с этим игроком; выбранный игрок немедленно Сбивается с ног (Knocked Down). Это не вызовет Перехода хода, если только сбитый игрок не владел мячом. Если у этого игрока есть навыки Claws или Mighty Blow, он обязан использовать их при Броске на Пробитие Брони для Сбитого с ног игрока.

Если у этого игрока выпадает 1–3, а рядом нет Стоящих на ногах товарищей по команде, то он получает состояние Distracted.
