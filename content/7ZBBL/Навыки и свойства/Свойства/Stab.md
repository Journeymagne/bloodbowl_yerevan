Когда этот игрок активирован, он может объявить особое действие Stab; нет ограничения на количество игроков, которые могут объявить это особое действие за каждый ход.

Когда этот игрок выполняет особое действие Stab, выберите стоящего на ногах игрока противника, соседнего с этим игроком, и сделайте бросок на Пробитие Брони для выбранного игрока. Этот бросок на Пробитие Брони не может быть модифицирован никаким образом. Если броня игрока пробита, сделайте для него Бросок на Травму, в противном случае ничего не происходит.

Этот игрок может использовать особое действие Stab вместо действия Блок, выполняемого в рамках действия Блиц.
