Когда этот игрок активирован, он может объявить особое действие Заблев; нет ограничения на количество игроков, которые могут объявить это особое действие в течение хода.

Когда этот игрок особое действие Заблев, выберите Стоящего на ногах игрока противника, соседнего с этим игроком, и бросьте D6. При результате 2+ этот игрок поражает игрока противника тугой струёй едкой блевотины. Сделайте Бросок на Пробитие Брони для целевого игрока. Этот Бросок на Пробитие Брони не может быть изменён никаким образом. Если Броня игрока пробита, сделайте для него Бросок на Травму, в противном случае ничего не происходит.

При результате 1 этот игрок заблевывает сам себя; сделайте Бросок на Пробитие Брони для этого игрока. Этот Бросок на Пробитие Брони не может быть изменён никаким образом. Если Броня этого игрока пробита, сделайте для него Бросок на Травму, в противном случае ничего не происходит.

Этот игрок может использовать особое действие Заблев вместо действия Блока, выполняемого в рамках действия Блиц, если пожелает.
