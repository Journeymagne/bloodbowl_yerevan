Каждый раз, когда игрок противника пытается Уклониться (Dodge) из клетки в Зоне контроля этого игрока, этот игрок может использовать данный навык.

Когда этот игрок использует этот навык, бросьте D6. При результате 1–3 ничего не происходит. При результате 4+ этот игрок немедленно перемещается в клетку, которую покинул игрок противника. Максимум применений этого навыка в ход, равен MA игрока.

Если игрок пытается покинуть Зону контроля нескольких игроков с этим навыком одновременно, только один из них может использовать этот навык.
