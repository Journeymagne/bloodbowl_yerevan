Когда этот игрок активирован, он может объявить особое действие Shiv;  это особое действие можно объявить один раз в ход команды.

Когда этот игрок выполняет особое действие Shiv, выберите стоящего на ногах игрока противника, соседнего с этим игроком, и сделайте бросок на Пробитие Брони для выбранного игрока. Этот бросок на Пробитие Брони не может быть модифицирован никаким образом. Если броня игрока пробита, сделайте для него Бросок на Травму, в противном случае ничего не происходит.

Этот игрок может использовать особое действие Shiv вместо действия Блок, выполняемого в рамках действия Блиц.
