Когда этот игрок получает Серьезную травму, киньте d6, на 1-3 ничего не происходит, на 4+, если результат травмы был Badly Hurt, то понизьте случайную характеристику игроку (по таблице выбора случайной характеристики) и верните его в резерв. При любой другой травме, просто считайте ее Badly Hurt. Этот навык нельзя выбрать если у игрока есть черта Regeneration. Этот навык могут брать только игроки с чертой Stunty.

| d8  | Характеристика |
| --- | -------------- |
| 1   | AV             |
| 2   | AV/PA          |
| 3-4 | AV/MA/PA       |
| 5   | MA/PA          |
| 6   | AG/MA          |
| 7   | AG/ST          |
| 8   | Любая
